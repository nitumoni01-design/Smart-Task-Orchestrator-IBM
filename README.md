# Smart Task Orchestrator (SoloSync)

This project is built using IBM watsonx Orchestrate.

It converts meeting notes into:
- Actionable tasks
- Deadlines
- Reminders

## Problem
Teams forget tasks discussed in meetings.

## Solution
An AI agent that reads meeting notes and automatically extracts tasks and creates reminders.

## Built With
- IBM watsonx Orchestrate
- Generative AI Prompt
- Agentic Workflow
